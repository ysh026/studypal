#  StudyPal - 你的 AI 沉浸式学习伴侣  

> ✨ **最新版本**: v1.1.0 (Stable)        
> 🔗 **在线使用**: [点击这里直接打开](https://ysh026.github.io/studypal/)        

##  关于这个项目
写这个项目的初衷是解决我自己的需求：我需要一个既能管理任务流，又能随时唤起 LLM 进行深度问答的 All-in-One 界面。

## ✨ 我最喜欢的功能
* **智能创建任务**：可以直接在聊天框跟 AI 说“我要复习30分钟英语”，它会自动识别意图，帮你在左边**自动创建任务**（可防重）。当然，你也可以手动创建。
* **AI 主动督导**：番茄钟倒计时结束后，AI会**主动发消息**来考核你的学习成果（比如：“复习得怎么样？考考你...”）。
* **支持AI预设自定义**：支持 **“人设切换”**，你可以让它扮成“严厉的考研助手”或者“苏格拉底式导师”。支持 Markdown，代码、论文格式渲染都很漂亮。
* **任务清单文件夹**：支持文件夹分类，任务多了可以**批量选择、移动、删除**，界面使用简洁风格。
* **绝对安全**：这是一个纯静态网页，**没有后台服务器**。你的聊天记录、API Key 全部存在你**自己的浏览器**里。

##  如何使用
1.  点击上面的 **在线使用** 链接。
2.  点击左下角的 **⚙️ 设置**。
3.  填入你的 API Key（支持 **硅基流动 / DeepSeek / OpenAI / Gemini /自定义**）。          
    * *如果你没有 Key，可以用硅基流动（SiliconFlow）有免费额度（能用我的邀请链接我会很开心的🥹），国内直连速度快。*
    * 硅基流动注册链接：https://cloud.siliconflow.cn/i/dNmJ6W0N    
4.  开始聊天或计时！

## 📝 更新日志 (Changelog)

### v1.1.0 (2025-11-25) - 对话掌控力升级
大幅增强了 AI 聊天的交互体验。
*  **停止生成**：新增停止按钮，可随时中断 AI 的输出。
*  **重新生成**：对不满意的回答一键重试（仅限最后一条回复）。
*  **单条删除**：支持单独删除某一条对话气泡，保持上下文整洁。
*  **多行输入**：输入框升级为弹性高度文本域，支持 `Shift+Enter` 换行。

### v1.0.0 - 正式版 (里程碑更新)
集成了 v0.9.x 后期的所有核心打磨，标志着产品交互与视觉的成熟。
* **沉浸专注模式**：
    * 新增全屏专注入口，支持 **日夜间模式 (☀️/🌙)** 切换。
    * 引入 **Hero 动画**，进入/退出时拥有丝滑的缩放过渡效果。
* **拟物化滚轮**：计时器摒弃滑条，采用**垂直无限滚轮**，支持上下滑动快速选时。
* **极致 UI/UX**：  
    * 全面按照**简约设计**规范重绘：毛玻璃特效、弥散阴影、微交互（按钮悬停上浮、点击缩放）。
    * 任务移动功能升级为 **中央弹窗 (Modal)**，彻底解决侧边栏布局挤压问题。
* **智能增强**：
    * **防重逻辑**：AI 自动添加任务时，会检测是否存在同名任务，避免重复。
    * **模型记忆**：不同服务商自动记忆上次选择的模型 ID。
* **💾 核心修复**：重构数据备份机制，修复了“恢复后刷新失效”的逻辑漏洞，实现热重载。

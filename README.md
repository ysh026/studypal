#  StudyPal - 你的 AI 沉浸式学习伴侣  

> ✨ **最新版本**: v0.9.7 (Stable)      
> 🔗 **在线使用**: [点击这里直接打开](https://ysh026.github.io/studypal/)    

##  关于这个项目

写这个项目的初衷是解决我自己的需求：我需要一个既能管理任务流，又能随时唤起 LLM 进行深度问答的 All-in-One 界面。这不是什么商业软件，就是我自己为了解决学习痛点手搓的工具，觉得好用分享给大家。

## ✨ 我最喜欢的功能
* **🗣️ 说句话就能建任务**：不用手动点来点去，直接在聊天框跟 AI 说“我要复习30分钟英语”，它会自动识别意图，帮你在左边**自动创建任务**（还能智能防重）。当然，你也可以手动创建。
* **🍅 AI 主动督导**：番茄钟倒计时结束后，AI会**主动发消息**来考核你的学习成果（比如：“复习得怎么样？考考你...”）。
* **🧠 它是真的懂你**：支持 **“人设切换”**，你可以让它扮成“严厉的考研教练”或者“苏格拉底式导师”。支持 Markdown，代码、论文格式渲染都很漂亮。
* **📂 强迫症福音**：支持文件夹分类，任务多了可以**批量选择、移动、删除**，界面使用简洁风格。
* **🔒 绝对安全**：这是一个纯静态网页，**没有后台服务器**。你的聊天记录、API Key 全部存在你**自己的浏览器**里，隐私完全掌握在自己手中。
